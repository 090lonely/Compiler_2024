# 语法分析器
## 两种分析方式优劣
1.如果可以用工具生成（如 YACC、ANTLR），Bottom-up 分析器更方便。
如果需要手写，Top-down 分析器更简单直观。
2.学习阶段：Top-down 更容易，便于理解语法分析原理。
生产环境：Bottom-up 分析器更强大且通用，但依赖工具生成。
